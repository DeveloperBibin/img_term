Image 2 ANSI
=
Converts an image to a representation in ANSI

Installation
-
1. Copy the repository
2. Run:
```bash
pip install opencv-python numba numpy
```
Usage
-

Display an image in terminal 
```bash
./img_term.py img.jpg 
```
Render your video capture device to terminal:
```bash
./img_term.py
```
